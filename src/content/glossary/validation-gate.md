term: Validation gate

definition: >-
  One check in the five-gate chain every generated scene must pass before it can play: strict
  type-checking, an allow-list lint, a monome-playability marker check, and a headless render
  smoke test. A bounded self-repair loop retries a failing scene before giving up.
